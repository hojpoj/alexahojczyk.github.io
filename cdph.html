CDPH Work Overview

  • Platforms Covered: Responsive Web
  • Target Users: General users across all offices for client included nurse practitioners, clinicians, doctors, genetic counselors, expecting moms, new moms, general public looking for public health guidance (i.e. COVID guidelines), etc

These projects were all covered within one of the largest engagements with California. During this time, we released two new major applications that covered over 20,000+ users as well as maintained and improved the user interface and user experience within multiple pre-existing applications which affected over 500,000 users at any given time.
