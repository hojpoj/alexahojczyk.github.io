External API Portal Overview

  • Company: Fifth Third Bank
  • Platforms Covered: Responsive Web
  • Target Users: Fintech developers/developers looking to leverage a banking API as well as Fintech upper management looking to leverage a banking API
  • Deliverables: Low fidelity wireframes created in Sketch, User research plans, User research synthesis report, Competitive analysis of major API portals on the market, User personas for major user groups, Style guide and design system assets, Accessibility recommendations/considerations

Within Enterprise applications at Fifth Third, I led all user experience and user interfaces efforts over the past year within several major projects. One of these projects was aimed at creating an external facing API portal for the bank. During this time, we conducted major competitive and user research around API portals and API integration. The end product was a fully fleshed out release plan with three phases, wireframes, site maps, style guides, branding resources, etc supplied for each phase.
